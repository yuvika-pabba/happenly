# 🎉 Happenly — Supabase + Streamlit Event Planner

An intelligent event planning platform that simplifies creating, organizing, and managing events using a modern full-stack web architecture powered by **Streamlit** and **Supabase**.

---

## ✨ Overview

Happenly is designed to remove the friction from event planning by providing a clean, interactive interface where users can create events, manage RSVPs, and track attendees in real time.

Built with a focus on simplicity, scalability, and real-time data synchronization.

---

## 🚀 Features

- 🔐 Secure user authentication (Supabase Auth)
- 📅 Create, edit, and manage events dynamically
- 🙋 RSVP system for attendees
- 📊 Real-time database updates
- 🎨 Clean and responsive Streamlit UI
- ☁️ Cloud-based PostgreSQL database (Supabase)
- ⚡ Fast Python-powered frontend

---

## 🧰 Tech Stack

- **Frontend:** Streamlit  
- **Backend & Database:** Supabase (PostgreSQL)  
- **Language:** Python  
- **Authentication:** Supabase Auth  
- **Hosting:** Streamlit Cloud / Local deployment

## ⚙️ Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/yuvika-pabba/happenly.git
cd happenly
```

### 2. Create virtual environment (recommended)
```
python -m venv venv

# Mac/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 3. Install dependencies
```
pip install -r requirements.txt
```

### 4. Set environment variables
Create a .env file in the root directory:
```
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_anon_key
```

### 5. Run the app
```
streamlit run app.py
```
